基本介绍

- 适配器模式(Adapter Pattern)将某个类的接口转化成客户端期望的另一种接口表示，主要目的是兼容性，让原本因接口不匹配不能一起工作的两个类可以协同工作。其别名为包装器(Wapper)
- 适配器模式属于结构型模式
- 主要分为三类：类适配器模式、对象适配器模式、接口适配器模式
